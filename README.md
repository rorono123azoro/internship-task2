# Internship Task 2: E-commerce Landing Page

## Overview
A fully responsive, modern e-commerce landing page constructed using core web technologies. The layout features an interactive shopping aesthetic aimed at creating high user engagement and demonstrating core UI layout skills.

## Key Features
- **Interactive Toggles**: Dynamic UI components managed by Vanilla JavaScript.
- **Responsive Layout**: Designed to adapt cleanly across multiple device dimensions utilizing standard CSS methodologies.
- **Form Validation**: Interactive Javascript logic to ensure proper data input formats.
- **Modern UI Elements**: Usage of custom image assets and advanced CSS positioning.

## Technologies Used
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Folder Structure
```text
/
├── assets/        - Assorted images and UI graphics (e.g., premium_headphones.png)
├── index.html     - Main layout
├── styles.css     - Styling for product cards, navigation, and layout
└── script.js      - Dynamic logical functions and toggles
```

## How to Run Locally
1. Clone the repository to your system.
2. Browse the directory and simply open `index.html` in your favorite web browser.
